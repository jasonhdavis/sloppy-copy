# 004.5 — AI Provider Integration Verification

Verifies the AI provider abstraction defined in [`tasks/004-ai-provider-layer.md`](tasks/004-ai-provider-layer.md).

## A. Intent

Confirm that:

- All AI calls route through [`AIClient.generate()`](ai/client.py:1)
- Prompts load from `/prompts`
- Providers can switch via `DEFAULT_AI_PROVIDER`
- Responses are strict JSON and parsed to Python dictionaries
- Retry logic activates on invalid JSON

---

## B. Preconditions

- Django project runs
- `.env` contains required keys
- `/prompts` directory exists
- AI client module present

Environment variables required:

```
OPENAI_API_KEY
ANTHROPIC_API_KEY
DEFAULT_AI_PROVIDER
```

---

## C. Environment Verification

Run shell:

```
python manage.py shell
```

```
import os

os.getenv("OPENAI_API_KEY")
os.getenv("ANTHROPIC_API_KEY")
os.getenv("DEFAULT_AI_PROVIDER")
```

Expected:

- non‑empty values
- provider equals `openai` or `anthropic`

---

## D. Prompt Loading Test

Prompts must load from `/prompts`.

```
from ai.client import AIClient

client = AIClient()
client.load_prompt("evaluate_story")
```

Expected result:

```
{
  "system": "...",
  "user": "..."
}
```

---

## E. Basic AI Execution Test

```
from ai.client import AIClient

client = AIClient()

response = client.generate(
  prompt="Return JSON {\"hello\": \"world\"}",
  system="Respond only with JSON",
  model="gpt-4o",
  temperature=0
)
```

Expected:

```
{"hello": "world"}
```

Verify type:

```
isinstance(response, dict)
```

---

## F. Provider Switching Test

Switch provider via environment variable.

Example:

```
export DEFAULT_AI_PROVIDER=openai
```

Run AI test again.

Then:

```
export DEFAULT_AI_PROVIDER=anthropic
```

Repeat test.

Expected:

- same client interface works
- provider changes internally

---

## G. JSON Enforcement Test

AI responses must be JSON only.

If response contains text around JSON, parsing should fail and retry.

Expected behavior:

1. invalid JSON detected
2. retry issued
3. valid JSON returned

---

## H. Retry Behavior

Retry conditions:

- invalid JSON
- network errors
- rate limits

Backoff schedule:

```
1s
2s
4s
```

Maximum attempts: 3.

---

## I. Logging

Verify logs include:

- provider
- model
- latency
- retries
- prompt name

Example log:

```
AI_REQUEST provider=openai prompt=evaluate_story retries=0
```

## J. Pass Criteria

PASS

- prompts load
- AIClient returns dict
- provider switching works
- retries function
- JSON enforced

FAIL

- direct provider SDK usage detected
- JSON not parsed
- provider switch requires code change
