# 007.5 — Rewrite Generation Verification

Verifies that the rewrite engine defined in [`tasks/007-rewrite-generation-engine.md`](tasks/007-rewrite-generation-engine.md) correctly generates rewritten articles and stores them deterministically.

Primary service: [`RewriteGenerationService`](rewrites/services.py:1)
AI entrypoint: [`AIClient.generate()`](ai/client.py:1)

---

## A. Preconditions

Stories must exist with:

```
status="matched"
```

Verify:

```
python manage.py shell
```

```
from stories.models import Story
Story.objects.filter(status="matched").count()
```

Expected:

```
> 0
```

---

## B. CLI Verification

Run rewrite generation:

```
python manage.py generate_rewrites --limit 3
```

Expected console output:

```
Generating 3 rewrites
Story <id> rewritten
```

CLI handler invokes:

[`RewriteGenerationService.generate_batch()`](rewrites/services.py:1)

---

## C. Database Validation

Check rewrite rows:

```
from rewrites.models import Rewrite
Rewrite.objects.values(
 "id",
 "story",
 "headline"
)
```

Expected properties:

• `headline` non‑empty string

• `body` long‑form text

• linked `story`

---

## D. Story State Validation

Verify state transition:

```
Story.objects.filter(status="rewritten")
```

Expected:

• stories previously `matched` now `rewritten`

---

## E. JSON Output Validation

Expected structure from [`AIClient.generate()`](ai/client.py:1):

```
{
 "headline": "string",
 "body": "string"
}
```

Invalid JSON handling:

• retry once

• if failure persists story remains `matched`

---

## F. Idempotency Test

Run twice:

```
python manage.py generate_rewrites --limit 5
python manage.py generate_rewrites --limit 5
```

Second run expected:

```
0 stories processed
```

Selector reference:

[`get_matched_stories()`](stories/selectors.py:1)

---

## G. Failure Tests

### AI Provider Failure

Simulate invalid API key.

Expected:

• story skipped

• status remains `matched`

---

### Malformed Output

Force invalid JSON from test provider.

Expected behavior:

1. retry once
2. log error
3. story not transitioned

---

## H. Performance Check

Run:

```
python manage.py generate_rewrites --limit 20
```

Expected:

• sequential processing

• predictable runtime
