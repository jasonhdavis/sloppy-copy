# 003.5 — Feed Ingestion Verification

Verifies the RSS ingestion system defined in [`tasks/003-feed-ingestion-system.md`](tasks/003-feed-ingestion-system.md).

## A. Intent

Confirm that:

- RSS feeds are fetched and parsed
- `Story` rows are created
- `status = "ingested"`
- duplicate URLs are not inserted
- ingestion is idempotent

---

## B. Preconditions

```
python manage.py migrate
python manage.py check
```

At least one Source must exist.

---

## C. Test Data Setup

```
python manage.py shell
```

```
from sources.models import Source

Source.objects.create(
    name="Test Feed",
    rss_url="https://hnrss.org/frontpage",
    is_active=True
)
```

---

## D. Execute Ingestion

```
python manage.py ingest_feeds
```

Expected:

- feed fetched
- entries parsed
- stories inserted

---

## E. Database Assertions

```
python manage.py shell
```

```
from stories.models import Story

Story.objects.count()
Story.objects.filter(status="ingested").count()
```

Expected:

- count > 0
- all stories status=ingested

---

## F. Idempotency Test

Run ingestion again:

```
python manage.py ingest_feeds
```

Expected:

- no duplicate URLs

Verify:

```
from django.db.models import Count
Story.objects.values("url").annotate(c=Count("id")).filter(c__gt=1)
```

Expected result:

Empty queryset.

---

## G. Normalization Checks

Verify fields:

- title
- url
- summary
- published_at

```
Story.objects.filter(title="").count()
Story.objects.filter(url="").count()
```

Expected:

0

---

## H. Error Handling

Create invalid source:

```
Source.objects.create(
    name="Broken",
    rss_url="https://invalid.example/rss",
    is_active=True
)
```

Run ingestion.

Expected:

- error logged
- other feeds continue

---

## I. Pass Criteria

PASS

- stories created
- status=ingested
- duplicates prevented
- invalid feeds do not crash

FAIL

- duplicate URLs
- command crash
- missing required fields
